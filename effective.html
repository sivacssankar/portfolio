Dish Destiny

Dish Destiny is an Android application developed using Android Studio and Kotlin, designed to help users discover and create delicious recipes based on available ingredients. The app offers a smart recipe recommendation system, where users can input ingredients they have, and it suggests suitable recipes. It also provides step-by-step cooking instructions, nutritional information, and video tutorials for better guidance. Users can save favorite recipes, create shopping lists, and share recipes with friends. With an intuitive interface and personalized recommendations, Dish Destiny makes cooking more convenient and enjoyable for all food lovers.
